# Gestalt Security Framework
The Gestalt Security Framework (GSF) is an open source framework that provides navigation and mapping across multiple IT security controls frameworks (e.g. NIST 800-53, ISO 27000 series, PCI-DSS, COBIT, CIS, ACSC (IRAP), HIPAA, SOC2).
